# World Happiness Report - Data Analysis and Visualization

Bu proje, Dünya Mutluluk Raporu veri seti kullanılarak ülkelerin mutluluk düzeylerini etkileyen faktörlerin analiz edilmesi amacıyla geliştirilmiştir.

## Proje Özeti
Proje kapsamında aşağıdaki veri bilimi süreçleri uygulanmıştır:
* **Veri Temizleme:** Eksik verilerin (missing values) tespiti ve uygun yöntemlerle doldurulması.
* **Keşifsel Veri Analizi (EDA):** Veri setinin istatistiksel özetlerinin çıkarılması.
* **Veri Görselleştirme:** Mutluluk skoru ile GSYİH, sosyal destek ve yaşam beklentisi gibi değişkenler arasındaki ilişkilerin Matplotlib ve Seaborn kütüphaneleriyle görselleştirilmesi.

## Kullanılan Teknolojiler
* Python
* Pandas & NumPy
* Matplotlib & Seaborn

## Nasıl Çalıştırılır?
1. Bu depoyu klonlayın.
2. `jupyter notebook` ortamında `Data_Project.ipynb` dosyasını açın ve hücreleri çalıştırın.
